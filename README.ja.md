PromptTapは、AIエージェントとの対話を効率化するために設計されたmacOSネイティブアプリケーションです。プロンプトの下書き、管理、およびMac上のあらゆるアプリケーションへの即時入力をサポートする専用のワークスペースを提供します。

## 高速なワークフロー

PromptTapはスピードを重視して構築されています。UIではなく、思考に集中できます：

1. **呼び出し**: 任意のアプリの入力フィールドで **Command** キーを2回押すと、PromptTapが表示されます。
2. **下書き**: シンプルで集中を妨げないエディタでプロンプトを入力します。
3. **送信**: **Cmd + S** を押すと、自動的に前のアプリに戻り、プロンプトが貼り付けられます。

## 主な機能

- **グローバルホットキーによるアクセス**: シンプルなショートカット（デフォルト：**Command** 2回押し）で、どこからでも即座にPromptTapを開けます。
- **ダイレクト送信**: 1つのコマンドで、直前まで使用していたアプリケーションにプロンプトをシームレスに貼り付けます。
- **スマート履歴**: 過去のプロンプトを自動的に追跡し、再利用や改善を容易にします。テンプレートやリザーブ（予約語）への未保存の変更も履歴に保存されます。
- **テンプレートとリザーブ**: 頻繁に使用する構造を整理できます。グローバル検索を使用して、即座に見つけて適用できます。
- **Vimモードのサポート**: パワーユーザー向けに、CodeMirror 6を使用した本格的なVimキーバインドモードを利用可能です。
- **ネイティブな体験**: macOSのデザインパターンとシステム設定を尊重した、SwiftUIによるクリーンな2ペインレイアウト。

## キーボードショートカット

| ショートカット | アクション |
| --- | --- |
| `Command` 2回 | PromptTapを開く / ターゲットアプリに戻る |
| `Cmd + S` | **送信 (Submit)**: 前のアプリに戻ってプロンプトを貼り付ける |
| `Cmd + C` | **コピー**: プロンプト全文をクリップボードにコピー |
| `Cmd + F` | **グローバル検索**: テンプレート、リザーブ、履歴を横断検索 |
| `Cmd + N` | **次のアイテム**: サイドバーのアイテムを順に切り替え |
| `Cmd + P` | **前のアイテム**: サイドバーのアイテムを逆順に切り替え |
| `Cmd + H` | **履歴切り替え**: 最新の履歴と現在のプロンプトを切り替え |
| `Cmd + T` | **テンプレート検索**パネルを開く |
| `Cmd + R` | **リザーブ検索**パネルを開く |
| `Cmd + E` | **エディタ**にフォーカス |
| `Cmd + L` | **サイドバーリスト**にフォーカス |
| `Cmd + ,` | **設定**を開く |

## インストール

### Homebrew を使用する場合 (推奨)

```bash
brew tap tokorom/tap
brew install --cask prompttap
```

### 手動インストール

1. [Releases](https://github.com/tokorom/PromptTap/releases) ページから最新バージョンの `PromptTap-x.x.x.dmg` をダウンロードします。
2. `PromptTap-x.x.x.dmg` を開き、 `PromptTap.app` を `/Applications` フォルダにドラッグします。

## アクセシビリティの権限

初回起動時に、**アクセシビリティ**の権限を許可するよう求められます。これは以下の機能に必要です：
- 「送信」を有効にするための、直前のアプリケーションの検出。
- 他のアプリへのテキストの自動貼り付け。
- グローバルホットキーの検出。

**注意**: アプリの起動中に権限を許可した場合は、設定を反映させてグローバルホットキーを有効にするために、**PromptTapを再起動**してください。

## 設定

**設定** (`Cmd + ,`) では、以下をカスタマイズできます：
- **グローバルホットキー**: 2回押しのトリガーを変更したり、カスタムの組み合わせを設定したりできます。
- **エディタ設定**: Vimキーバインドの切り替えや行の折り返し設定。
- **履歴**: 保持する履歴アイテムの最大数を設定。
- **ストレージ**: テンプレートとリザーブをプレーンな `.txt` ファイルとして保存するカスタムフォルダを選択でき、同期や外部エディタでの編集が容易になります。
